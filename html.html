Sobre o HTML

O que é o HTML

HTML é uma linguagem de marcaçao, ou seja, fornece instruçoes do que deve ser mostrado na página. Para isso, a lingua usa um conjunto pre definido de elementos, que por sua vez, possui diversas tags com o objetivo de expressar um determinado conteudo

Benefícios:

  • Melhora a performance da página da web e a navegaçao fica mais rapida
  • Ela é aceita em todos os dispositivos, inclusive dispositivos moveis
  • Ela é aceita em diversos Browsers
  • Armazenamento de dados offline
  • Permite utilizar canvas
